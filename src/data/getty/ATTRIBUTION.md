# Getty AAT Attribution

This application contains information from the **Art & Architecture Thesaurus (AAT)**,
maintained by the Getty Research Institute.

AAT data is made available under the
[ODC Attribution License (ODC-By 1.0)](https://opendatacommons.org/licenses/by/1.0/).

**Source:** https://vocab.getty.edu/

**Usage:** Bundled vocabulary data in this directory was extracted from the
Getty AAT SPARQL endpoint (`https://vocab.getty.edu/sparql`) for offline use.
Each term retains its canonical AAT URI for linked data interoperability.

**Facets included:**
- Object Types / Work Types (from Object Genres, Visual Works, Information Forms hierarchies)
- Materials (from Materials hierarchy)
- Techniques / Processes (from Processes and Techniques hierarchy)
- Styles and Periods (from Styles and Periods hierarchy)
